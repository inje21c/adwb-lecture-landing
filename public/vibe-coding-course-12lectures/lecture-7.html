LECTURE 07
화면정의서 생성 프롬프트
다음 웹 도구의 화면정의서를 작성해줘. 화면은 하나의 메인 화면으로 구성된다. 상단에는 제목과 설명 문구가 있고, 중간에는 입력 폼이 있으며, 하단에는 생성 결과가 카드 형태로 표시된다. 결과 카드에는 각각 복사 버튼이 있다. 화면정의서에는 다음을 포함해라. - 화면 ID - 화면명 - 화면 목적 - 주요 구성요소 - 사용자 행동 - 시스템 반응 - 빈 상태 - 오류 상태 - 버튼/라벨 문구
