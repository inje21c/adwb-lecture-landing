## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 원하는 강의를 클릭합니다.
3. 슬라이드는 `←`, `→`, `Space`, `Home`, `End` 키로 이동합니다.
4. 브라우저 인쇄 기능을 사용하면 PDF로 저장할 수 있습니다.

## 설계 원칙

- 공통 실습으로 수업 안정성을 확보합니다.
- 개인 프로젝트로 각자의 업무 문제에 적용합니다.
- 실제 회사 데이터, 개인정보, API Key는 실습 자료에 포함하지 않습니다.
- Part 3는 데이터·AI·배포·보안의 업무 적용 경계를 명확히 설명합니다.
