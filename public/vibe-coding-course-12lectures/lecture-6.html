LECTURE 06
구현 프롬프트 만들기
나는 5강에서 만든 화면 목업을 기준으로 Working Backwards 1페이지 생성기를 구현하고 싶다. 이번 버전은 로컬 브라우저에서 실행되는 단일 HTML 파일이어야 한다. 서버, DB, 로그인, 외부 API는 사용하지 마라. HTML, CSS, JavaScript를 index.html 한 파일에 모두 포함해라. 필요 기능은 다음과 같다. 1. 아이디어, 사용자, 현재 불편, 기대 효과, 핵심 기능, KPI, 제약 조건 입력 2. 생성 버튼 3. 보도자료 문장, 고객 가치, KPI, 화면 프롬프트, Codex 프롬프트 출력 4. 각 결과 복사 버튼 5. 전체 초기화 버튼 6. 민감정보 입력 금지 안내 우선 1차 버전을 만들어라.
